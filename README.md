# ربات سوال و جواب ناشناس تلگرام

یک ربات تلگرام کامل برای سیستم سوال و جواب ناشناس با استفاده از Aiogram 3.x

## ویژگی‌ها

### 🔐 سیستم ناشناس
- ارسال سوالات به صورت کاملاً ناشناس
- پاسخ‌دهی ناشناس به سوالات
- عدم نمایش اطلاعات شخصی کاربران

### 📋 منوی اصلی
- ✍️ ارسال سوال ناشناس
- 👀 دیدن سوالات دیگران
- 💬 پاسخ دادن به سوالات
- ⭐ محبوب‌ترین پاسخ‌ها

### 📂 دسته‌بندی‌ها
- ❤️ روابط عاطفی
- 💼 کار و شغل
- 📚 تحصیل
- 🩺 سلامت
- 🎭 سایر

### 💬 سیستم پاسخ‌دهی
- پاسخ‌های متعدد به هر سوال
- سیستم رای‌دهی (👍/👎)
- نمایش بهترین پاسخ‌ها در ابتدا
- سنجاق کردن پاسخ‌های ویژه

### 🔧 پنل مدیریت
- حذف سوالات و پاسخ‌های نامناسب
- سنجاق کردن پاسخ‌های مهم
- آمار کلی سیستم

### 📊 پایگاه داده
- SQLite برای شروع (قابل ارتقا به PostgreSQL)
- جداول: users, questions, answers, votes
- مدیریت کامل داده‌ها

## نصب و راه‌اندازی

### 1. نصب وابستگی‌ها
```bash
pip install -r requirements.txt
```

### 2. تنظیم متغیرهای محیطی
فایل `.env` را ایجاد کنید:
```env
BOT_TOKEN=your_bot_token_here
ADMIN_IDS=123456789,987654321
DATABASE_URL=sqlite:///bot.db
```

### 3. اجرای ربات
```bash
python main.py
```

## ساختار پروژه

```
├── main.py              # فایل اصلی ربات
├── config.py            # تنظیمات و متون فارسی
├── database.py          # مدیریت پایگاه داده
├── handlers.py          # هندلرهای ربات
├── keyboards.py         # کیبوردهای اینلاین
├── states.py            # حالت‌های FSM
├── requirements.txt     # وابستگی‌ها
├── .env.example         # نمونه فایل محیطی
└── README.md           # مستندات
```

## استفاده

### برای کاربران عادی:
1. `/start` - شروع ربات
2. انتخاب "ارسال سوال ناشناس"
3. انتخاب دسته‌بندی
4. نوشتن سوال
5. مشاهده و پاسخ‌دهی به سوالات دیگران

### برای مدیران:
1. `/admin` - دسترسی به پنل مدیریت
2. مدیریت محتوا
3. حذف مطالب نامناسب
4. سنجاق کردن پاسخ‌های مهم

## ویژگی‌های فنی

- **زبان**: Python 3.8+
- **فریمورک**: Aiogram 3.x
- **پایگاه داده**: SQLite (قابل ارتقا)
- **حالت‌ها**: FSM برای مدیریت جریان کاربر
- **صفحه‌بندی**: برای نمایش لیست‌ها
- **مدیریت خطا**: هندلینگ کامل خطاها
- **لاگ‌گیری**: ثبت کامل فعالیت‌ها

## امنیت و حریم خصوصی

- تمام پیام‌ها به صورت ناشناس ذخیره می‌شوند
- هیچ اطلاعات شخصی کاربران نمایش داده نمی‌شود
- سیستم رای‌دهی محدود به یک رای per کاربر
- مدیریت محتوا توسط ادمین‌ها
